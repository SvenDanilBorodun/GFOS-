## Einfuhrung

Das Ideenbrett ist eine Plattform fur Innovationsmanagement. Hier konnen Sie:

- **Ideen einreichen:** Teilen Sie Ihre innovativen Vorschlage mit dem Team
- **Zusammenarbeiten:** Diskutieren und verbessern Sie Ideen gemeinsam
- **Abstimmen:** Zeigen Sie Unterstutzung durch "Gefallt mir" und Umfragen
- **Fortschritt verfolgen:** Beobachten Sie, wie Ideen umgesetzt werden
- **Belohnungen verdienen:** Sammeln Sie Erfahrungspunkte und Abzeichen

### Benutzerrollen

| Rolle | Beschreibung |
|-------|--------------|
| **Mitarbeiter** | Kann Ideen einreichen, kommentieren und abstimmen |
| **Projektleiter** | Kann zusatzlich den Status von Ideen andern |
| **Administrator** | Hat vollen Zugriff auf alle Funktionen |

## Erste Schritte

### Registrierung

Falls Sie noch kein Konto haben, konnen Sie sich selbst registrieren:

1. Offnen Sie die Anwendung im Browser
2. Klicken Sie auf **"Registrieren"** unter dem Anmeldeformular

![Screenshot: Registrierungsseite]
*Platzhalter fur Screenshot der Registrierungsseite*

3. Fullen Sie das Formular aus:
   - **Benutzername:** Wahlen Sie einen eindeutigen Namen
   - **E-Mail-Adresse:** Geben Sie Ihre Firmen-E-Mail ein
   - **Kennwort:** Mindestens 8 Zeichen
   - **Kennwort bestatigen:** Wiederholen Sie das Kennwort

4. Klicken Sie auf **"Registrieren"**

Nach erfolgreicher Registrierung werden Sie automatisch angemeldet.

### Anmeldung

1. Offnen Sie die Anwendung unter **http://localhost:3000**
2. Geben Sie Ihren **Benutzernamen** oder Ihre **E-Mail-Adresse** ein
3. Geben Sie Ihr **Kennwort** ein
4. Klicken Sie auf **"Anmelden"**

![Screenshot: Anmeldeseite]
*Platzhalter fur Screenshot der Anmeldeseite*

**Tipp:** Ihre Anmeldung bleibt aktiv, bis Sie sich abmelden oder den Browser schliessen.

### Abmeldung

1. Klicken Sie auf Ihr Profilbild oben rechts
2. Wahlen Sie **"Abmelden"** aus dem Menu

## Ideen

### Ideen durchsuchen

Klicken Sie auf **"Ideen"** in der Navigation, um alle Ideen zu sehen.

![Screenshot: Ideenubersicht]
*Platzhalter fur Screenshot der Ideenubersicht*

#### Filtern und Sortieren

Sie konnen Ideen nach verschiedenen Kriterien filtern:

**Nach Kategorie:**
- Alle
- Prozessoptimierung
- Produktinnovation
- Digitalisierung
- Nachhaltigkeit
- Sonstiges

**Nach Status:**
- Alle
- Konzept (neue Ideen)
- In Bearbeitung
- Abgeschlossen

**Sortieren nach:**
- Neueste zuerst
- Alteste zuerst
- Meiste "Gefallt mir"
- Wenigste "Gefallt mir"

**Suche:**
Geben Sie Stichworter in das Suchfeld ein, um Ideen nach Titel oder Beschreibung zu finden.

### Idee erstellen

1. Klicken Sie auf **"Neue Idee"** (oder das Plus-Symbol)
2. Fullen Sie das Formular aus:

![Screenshot: Idee erstellen]
*Platzhalter fur Screenshot des Formulars "Idee erstellen"*

| Feld | Beschreibung | Pflicht |
|------|--------------|---------|
| **Titel** | Kurzer, pragnanter Titel | Ja |
| **Beschreibung** | Ausfuhrliche Erlauterung der Idee | Ja |
| **Kategorie** | Themenbereich der Idee | Ja |
| **Schlagworter** | Stichworter zur besseren Auffindbarkeit | Nein |
| **Checkliste** | Aufgaben zur Umsetzung | Nein |

3. Klicken Sie auf **"Idee einreichen"**

**Tipp:** Eine gute Ideenbeschreibung enthalt:
- Das Problem, das gelost werden soll
- Die vorgeschlagene Losung
- Den erwarteten Nutzen

### Idee bearbeiten

Sie konnen Ihre eigenen Ideen jederzeit bearbeiten:

1. Offnen Sie die Idee durch Klicken auf den Titel
2. Klicken Sie auf **"Bearbeiten"** (Stift-Symbol)
3. Nehmen Sie Ihre Anderungen vor
4. Klicken Sie auf **"Speichern"**

**Hinweis:** Nur der Autor einer Idee kann diese bearbeiten. Administratoren konnen alle Ideen bearbeiten.

### Idee loschen

**Achtung:** Das Loschen einer Idee kann nicht ruckgangig gemacht werden!

1. Offnen Sie die Idee
2. Klicken Sie auf **"Loschen"** (Mulleimer-Symbol)
3. Bestatigen Sie die Loschung

**Hinweis:** Nur Administratoren konnen Ideen loschen.

### Idee als PDF exportieren

Sie konnen eine Idee als PDF-Dokument herunterladen:

1. Offnen Sie die Idee
2. Klicken Sie auf **"Als PDF exportieren"**
3. Die PDF-Datei wird automatisch heruntergeladen

Das PDF enthalt:
- Titel und Beschreibung
- Autor und Erstellungsdatum
- Status und Fortschritt
- Kommentare
- Checkliste

## Interaktion mit Ideen

### Gefallt mir

Zeigen Sie Ihre Unterstutzung fur eine Idee:

1. Klicken Sie auf das **Herz-Symbol** bei einer Idee
2. Das Symbol wird rot, wenn Ihnen die Idee gefallt
3. Klicken Sie erneut, um "Gefallt mir" zu entfernen

![Screenshot: Gefallt-mir-Funktion]
*Platzhalter fur Screenshot der Gefallt-mir-Funktion*

**Wochentliches Limit:**
Um faire Bewertungen zu gewahrleisten, haben Sie ein wochentliches Limit fur "Gefallt mir"-Vergaben. Das Limit wird jeden Montag zuruckgesetzt.

**Erfahrungspunkte:**
- Sie erhalten **10 EP** fur jedes "Gefallt mir", das Sie erhalten
- Der Autor der Idee wird benachrichtigt

### Kommentare

Diskutieren Sie Ideen in den Kommentaren:

1. Offnen Sie eine Idee
2. Scrollen Sie zum Kommentarbereich
3. Schreiben Sie Ihren Kommentar in das Textfeld
4. Klicken Sie auf **"Kommentar senden"**

![Screenshot: Kommentarbereich]
*Platzhalter fur Screenshot des Kommentarbereichs*

**Kommentar bearbeiten:**
1. Klicken Sie auf das Stift-Symbol bei Ihrem Kommentar
2. Bearbeiten Sie den Text
3. Klicken Sie auf **"Speichern"**

**Kommentar loschen:**
1. Klicken Sie auf das Mulleimer-Symbol bei Ihrem Kommentar
2. Bestatigen Sie die Loschung

**Erfahrungspunkte:**
- Sie erhalten **5 EP** fur jeden Kommentar, den Sie schreiben

### Reaktionen auf Kommentare

Sie konnen auf Kommentare mit Emojis reagieren:

1. Fahren Sie mit der Maus uber einen Kommentar
2. Klicken Sie auf das Emoji-Symbol
3. Wahlen Sie eine Reaktion:
   - Daumen hoch
   - Herz
   - Lachen
   - Uberrascht
   - Traurig

Die Reaktionen werden unter dem Kommentar angezeigt.

## Checklisten

Checklisten helfen dabei, den Fortschritt einer Idee zu verfolgen.

### Checkliste erstellen

Beim Erstellen oder Bearbeiten einer Idee:

1. Scrollen Sie zum Abschnitt **"Checkliste"**
2. Klicken Sie auf **"Aufgabe hinzufugen"**
3. Geben Sie den Aufgabentitel ein
4. Wiederholen Sie fur weitere Aufgaben

![Screenshot: Checkliste]
*Platzhalter fur Screenshot einer Checkliste*

### Aufgaben abhaken

1. Offnen Sie die Idee
2. Klicken Sie auf das Kontrollkastchen neben einer Aufgabe
3. Der Fortschrittsbalken aktualisiert sich automatisch

### Aufgaben entfernen

1. Bearbeiten Sie die Idee
2. Klicken Sie auf das X neben der Aufgabe
3. Speichern Sie die Anderungen

**Tipp:** Nutzen Sie Checklisten, um komplexe Ideen in uberschaubare Schritte aufzuteilen.

## Dateianhange

Sie konnen Dateien an Ideen anhangen, um Ihre Vorschlage zu verdeutlichen.

### Datei hochladen

1. Offnen Sie eine Idee oder erstellen Sie eine neue
2. Scrollen Sie zum Abschnitt **"Anhange"**
3. Ziehen Sie Dateien in den Bereich oder klicken Sie zum Auswahlen
4. Warten Sie, bis der Upload abgeschlossen ist

![Screenshot: Dateiupload]
*Platzhalter fur Screenshot des Dateiupload-Bereichs*

**Unterstutzte Dateitypen:**
- Bilder (PNG, JPG, GIF)
- Dokumente (PDF, DOC, DOCX, XLS, XLSX)
- Prasentationen (PPT, PPTX)
- Textdateien (TXT, CSV)

**Grossenlimit:** Maximal 10 MB pro Datei

### Datei herunterladen

1. Klicken Sie auf den Dateinamen im Anhang-Bereich
2. Die Datei wird heruntergeladen

### Datei entfernen

1. Bearbeiten Sie die Idee
2. Klicken Sie auf das X neben der Datei
3. Speichern Sie die Anderungen

## Gruppen

Jede Idee hat automatisch eine Diskussionsgruppe, in der Teammitglieder zusammenarbeiten konnen.

### Gruppe beitreten

1. Offnen Sie eine Idee
2. Klicken Sie auf **"Gruppe beitreten"**
3. Sie sind jetzt Mitglied der Gruppe

### Gruppenmitglieder anzeigen

1. Offnen Sie die Idee
2. Klicken Sie auf **"Gruppenmitglieder"**
3. Eine Liste aller Mitglieder wird angezeigt

**Mitgliedsrollen:**
- **Ersteller:** Der Autor der Idee (automatisch)
- **Mitglied:** Weitere Teilnehmer

### Gruppe verlassen

1. Offnen Sie die Idee
2. Klicken Sie auf **"Gruppe verlassen"**

**Hinweis:** Der Ersteller einer Idee kann die Gruppe nicht verlassen.

## Umfragen

Umfragen ermoglichen es, Meinungen einzuholen und Entscheidungen zu treffen.

### Umfrage erstellen

1. Klicken Sie auf **"Umfragen"** in der Navigation
2. Klicken Sie auf **"Neue Umfrage"**
3. Fullen Sie das Formular aus:

![Screenshot: Umfrage erstellen]
*Platzhalter fur Screenshot des Formulars "Umfrage erstellen"*

| Feld | Beschreibung |
|------|--------------|
| **Frage** | Die Frage, die beantwortet werden soll |
| **Beschreibung** | Zusatzliche Informationen (optional) |
| **Optionen** | Die Antwortmoglichkeiten (mindestens 2) |
| **Ablaufdatum** | Wann die Umfrage endet (optional) |
| **Anonym** | Ob die Stimmen anonym sein sollen |
| **Mehrfachauswahl** | Ob mehrere Optionen gewahlt werden konnen |

4. Klicken Sie auf **"Umfrage erstellen"**

### An Umfrage teilnehmen

1. Offnen Sie eine Umfrage
2. Wahlen Sie Ihre bevorzugte(n) Option(en)
3. Klicken Sie auf **"Abstimmen"**

**Hinweis:** Bei nicht-anonymen Umfragen konnen Sie sehen, wer wie abgestimmt hat.

### Umfrageergebnisse

Die Ergebnisse werden als Balkendiagramm angezeigt:
- Prozentsatz jeder Option
- Anzahl der Stimmen
- (Bei nicht-anonymen Umfragen) Teilnehmerliste

## Gamification

Das Ideenbrett belohnt aktive Teilnahme mit Erfahrungspunkten und Abzeichen.

### Erfahrungspunkte

Sie erhalten Erfahrungspunkte (EP) fur verschiedene Aktionen:

| Aktion | Erfahrungspunkte |
|--------|------------------|
| Idee einreichen | +50 EP |
| "Gefallt mir" erhalten | +10 EP |
| Kommentar schreiben | +5 EP |
| Idee abschliessen | +100 EP |

Mit steigenden Erfahrungspunkten erreichen Sie hohere Stufen:

| Stufe | Erforderliche EP | Bezeichnung |
|-------|------------------|-------------|
| 1 | 0 | Einsteiger |
| 2 | 100 | Mitdenker |
| 3 | 250 | Ideengeber |
| 4 | 500 | Innovator |
| 5 | 1000 | Vision√§r |
| 6 | 2000 | Meisterdenker |
| 7 | 3500 | Pionier |
| 8 | 5000 | Experte |
| 9 | 7500 | Guru |
| 10 | 10000 | Legende |

Ihre aktuelle Stufe wird in Ihrem Profil und neben Ihrem Namen angezeigt.

### Abzeichen

Verdienen Sie Abzeichen fur besondere Leistungen:

## Haufige Fragen

### Allgemein

**F: Kann ich meine eigene Idee bewerten?**
A: Nein, Sie konnen Ihre eigenen Ideen nicht mit "Gefallt mir" markieren.

**F: Wie kann ich mein Kennwort zurucksetzen?**
A: Kontaktieren Sie einen Administrator, um Ihr Kennwort zurucksetzen zu lassen.

**F: Warum kann ich nicht mehr "Gefallt mir" vergeben?**
A: Sie haben Ihr wochentliches Limit erreicht. Das Limit wird jeden Montag zuruckgesetzt.

### Ideen

**F: Kann ich eine Idee loschen?**
A: Nur Administratoren konnen Ideen loschen. Wenden Sie sich an einen Administrator.

**F: Was bedeuten die verschiedenen Status?**
A:
- **Konzept:** Neue Idee, noch nicht bearbeitet
- **In Bearbeitung:** Idee wird aktiv umgesetzt
- **Abgeschlossen:** Idee wurde vollstandig umgesetzt

**F: Wer kann den Status einer Idee andern?**
A: Der Autor der Idee, Projektleiter und Administratoren.

### Technisch

**F: Die Anwendung ladt sehr langsam. Was kann ich tun?**
A:
1. Uberprufen Sie Ihre Internetverbindung
2. Leeren Sie den Browser-Cache
3. Versuchen Sie einen anderen Browser

**F: Ich kann mich nicht anmelden. Was soll ich tun?**
A:
1. Uberprufen Sie Benutzername und Kennwort
2. Stellen Sie sicher, dass die Feststelltaste deaktiviert ist
3. Kontaktieren Sie einen Administrator

**F: Meine Benachrichtigungen funktionieren nicht.**
A: Stellen Sie sicher, dass Benachrichtigungen in Ihrem Browser erlaubt sind.
